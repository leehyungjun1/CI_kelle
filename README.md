# 프로젝트 기능 정리

이 프로젝트는 CodeIgniter 4 기반으로 구축된 **관리자·회원 통합 게시판 시스템**입니다.  
세션, 게시판, 스마트에디터, 컴포넌트 기반 구조 등을 체계적으로 작업중.

## 1. 세션 처리 (Admin / User)

### 관리자 세션
- `admin` 전용 세션 키 사용
- 관리자 영역 전용 필터 적용
- `/admin/*` 경로 접근 시 세션 인증 필수

### 사용자 세션
- 일반 사용자 `user` 세션 키 별도 분리
- 로그인/로그아웃 미들웨어 처리
- 사용자용 전용 페이지 접근 시 유저 세션 검증

### 세션 분리 설계 목적
- 관리자/유저 권한 혼동 방지
- 보안 강화
- 접근 제어 구조 명확화


## 2. 게시판 기본 기능

### 이메일 · 전화번호 · 주소 처리
- CodeIgniter Form Helper 기반 구현
- 폼 검증(Validation) 포함
- 사용자 입력 시 자동 정규화
- 이메일/전화번호 유효성 검사 적용

### 게시물 필터링 기능
- 검색 키워드 필터
- 날짜 범위 필터(시작일 ~ 종료일)
- 게시판 타입 필터
- 조건별 Builder 자동 적용
- 페이징 기능 포함


## 3. 이메일 컴포넌트

### 이메일 UI 컴포넌트 분리
- 이메일 입력 폼을 **컴포넌트 단위(View Fragment)**로 구현
- 재사용 가능한 구조
- 회원가입 / 프로필 수정 / 관리자 페이지 어디든 include 가능  

## 4. 스마트에디터 2 (SmartEditor2) 연동

### 이미지 업로드 구조
- 사진 업로드 시 **임시 폴더(temp)** 에 저장
- 실제 글 등록 시:
    - 게시판별 실제 저장 경로로 이동
    - 날짜 기반 디렉토리 구조 사용 (`/uploads/board/yyyy/mm/`)

### 임시 파일 자동 삭제 정책
- 하루 혹은 설정한 주기마다 temp 디렉토리 정리
- Cron 또는 CI4 Task로 자동 삭제 처리
- 서버 용량 관리 및 보안 강화 목적
